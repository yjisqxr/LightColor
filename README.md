# LightColor

---

**画板 + 实时3D光照渲染**

一个将2D绘画与3D光照渲染相结合的实时交互工具。在画板上绘制，右侧立即呈现带有光照效果的3D球体。

---

## ✨ 功能

- **实时绘制**：左侧画板自由绘画，支持鼠标和触摸
- **3D光照渲染**：右侧实时显示球体渲染结果
- **画笔控制**：调节颜色和大小
- **光源控制**：调节光源颜色、强度、环境光颜色和强度
- **渲染模式**：立体模式（逐像素光照）/ 平面模式（平均色+光照）
- **撤销/重做**：Ctrl+Z / Ctrl+Y，以每一笔为单位
- **一键填充**：用当前画笔颜色填充整个画板
- **采样颜色**：将画板平均色设为底色

---

## 🎮 操作说明

| 操作 | 说明 |
|:---|:---|
| 鼠标/触摸拖拽 | 在画板上绘制 |
| Ctrl + Z | 撤销上一笔 |
| Ctrl + Y | 重做被撤销的一笔 |
| 颜色选择器 | 更改画笔/光源/环境光颜色 |
| 滑块 | 调整画笔大小、光照强度、粗糙度 |
| 复选框 | 切换直接光/环境光/立体模式 |
| 按钮 | 清空画板、一键填充、采样颜色 |
